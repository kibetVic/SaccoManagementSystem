# SACCO Management System 
 
ASP.NET Core MVC application for managing SACCO operations using Database-First approach. 
 
## Features 
- Member Management 
- Contribution Tracking 
- Loan Management 
- Dashboard with Analytics 
- CRUD Operations 
 
## Technologies 
- ASP.NET Core 6.0 
- Entity Framework Core 
- SQL Server 
- Bootstrap 5 
- MVC Architecture 
 
## Setup Instructions 
 
1. Clone the repository: 
   ```bash 
   git clone https://github.com/kibetVic/SaccoManagementSystem.git 
   ``` 
 
2. Update connection string in `appsettings.json` 
 
3. Run the application: 
   ```bash 
   dotnet run 
   ```
